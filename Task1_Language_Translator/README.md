# 🌐 AI-Powered Language Translator (Task 1)

This project is developed as part of the CodeAlpha Artificial Intelligence Internship.

## Description
A GUI-based language translation application built using Python. It allows users to translate text into multiple languages using a translation API.

## Features
- User-friendly GUI using Tkinter
- Translate text into different languages
- Dark theme interface
- Copy translated text feature

## Technologies Used
- Python
- Tkinter
- Googletrans API

## How to Run
1. Install Python 3.10+
2. Install required library:
   pip install googletrans==4.0.0-rc1
3. Run the file:
   python translator.py
